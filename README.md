# Comandos para instalação Akira bot

$ pkg install git

$ git clone https://github.com/Tobizin2/unvicotor

$ cd Akira-bot

$ sh akira-bot

#Para o bot não cair 

Comandos!

$ pkg install pm2

$ npm i -g pm2

$ pm2 start index.js

$ pm2 monit

Caso finalize o termux e queira ativar novamente o pm2 basta digitar:

pm2 update

pm2 monit
